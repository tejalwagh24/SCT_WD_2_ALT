# UNKNOWN — Computational Entity

## Overview

UNKNOWN is an experimental computational experience created as an alternative implementation for SkillCraft Technology Task 2.

Unlike traditional calculator applications, UNKNOWN behaves like an unstable digital entity that reacts dynamically to user calculations.

---

## Features

- Live Computational Rendering
- Reactive System Environment
- Intelligent Entity Responses
- Dynamic Visual States
- Experimental UI/UX
- Computational Memory System
- Keyboard Interaction
- Error Handling
- Atmospheric Motion Design
- Responsive Layout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Concept

UNKNOWN explores the idea of transforming a basic calculator into an emotionally reactive computational system.

The interface changes behavior based on:
- complexity
- instability
- invalid patterns
- extreme calculations

creating an immersive interactive experience.
